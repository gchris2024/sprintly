# Sprintly

A W.I.P. productivity web application for managing personal goals, tracking daily todos, and reflecting on progress.

## Project Goals

- **Task Management**: Track daily todos with time estimates and completion status
- **Weekly Reflections**: Capture wins, challenges, and goals for continuous improvement
- **User Authentication**: Secure user accounts with JWT-based authentication
- **Sprint-based Workflow**: Organize work around weekly sprint cycles

## Technologies Used

### Backend

- **Node.js** with **Express** - REST API server
- **Prisma ORM** - Database modeling and migrations
- **PostgreSQL** - Relational database
- **JWT** - Authentication and authorization
- **Docker** - Containerization support

### Frontend

- **React** - UI framework
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server
- **React Router** - Client-side routing

## Motivation

This project is primarily for personal use. As an organized person, I wanted to build something that keeps everything all in one place. It's also a great learning experience for full-stack development.
